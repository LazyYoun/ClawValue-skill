# 🦞 ClawValue - OpenClaw 龙虾值评估系统 v4.0

量化你在 OpenClaw 上的 AI 自动化能力，生成趣味化的"龙虾能力估值"评估报告。

## ✨ 功能特点

- 📊 **使用深度评估**：五级技术指标体系（入门/初级/中级/高级/大师）
- 💰 **价值估算**：基于真实数据的价值量化，计算节省工时和 ROI
- 🦞 **趣味表达**："龙虾值"等级与调侃话术系统
- 🎨 **15 套主题风格**：深海龙虾、赛博霓虹、极简水墨等
- 🏆 **成就系统**：25+ 成就徽章，解锁你的专属称号
- 📈 **多维图表**：技能分布饼图、能力雷达图
- 🖼️ **图片生成**：AI 生成专属龙虾形象
- 🔗 **一键分享**：生成分享卡片，展示你的龙虾等级
- ⌨️ **快捷键支持**：R/S/T/E/G/? 等快捷操作

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动服务

```bash
python3 scripts/server.py --port 5002
```

访问：http://localhost:5002

### 命令行数据采集

```bash
python3 scripts/collector.py --output json
```

    ├── evaluation.py        # 评估引擎

## 🔧 技术栈

- **后端**: Python 3 + Flask
- **数据库**: SQLite
- **前端**: HTML/CSS/JavaScript + Chart.js
- **数据采集**: 解析 OpenClaw 日志和配置
- **图片生成**: 支持多种风格（像素、赛博、水墨等）
