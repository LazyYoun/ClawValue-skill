# 🦞 ClawValue 项目介绍

> OpenClaw 龙虾值评估系统 - 量化你的 AI 自动化能力

---

## 一、项目概述

### 1.1 产品定位

ClawValue 是一个 **OpenClaw 使用深度评估工具**，通过采集用户的日志、技能配置、会话数据，生成趣味化的「龙虾能力估值」评估报告。

**核心价值**：
- 📊 让用户直观了解自己对 OpenClaw 的使用深度
- 🎮 趣味化表达激发用户分享欲望
- 🦞 吸引更多用户使用 OpenClaw

### 1.2 目标用户

- OpenClaw 用户（主要）
- 对 AI 自动化工具感兴趣的开发者
- 喜欢分享趣味测试结果的用户

### 1.3 当前版本

| 属性 | 值 |
|------|------|
| 版本号 | v5.3 |
| 开发状态 | ✅ 已完成 |
| 主题数量 | 15 套 |
| 优化轮次 | 70 轮 |

## 二、建设需求

### 2.1 核心功能需求

#### 2.1.1 数据采集模块

| 模块 | 数据源 | 采集指标 |
|------|--------|----------|
| 日志解析 | `/tmp/openclaw/*.log` (JSONL) | 日志条目数、级别分布、工具调用、错误数量 |
| 技能扫描 | `~/.openclaw/workspace/skills/` | 技能总数、自定义技能数、分类分布 |
| 配置分析 | `~/.openclaw/openclaw.json` | Agent 数量、渠道列表、心跳状态 |

#### 2.1.2 评估引擎

**使用深度等级**：

| 等级 | 名称 | 分数范围 | 特征描述 |
|------|------|----------|----------|
| Lv.1 | 🐣 入门小白 | 0-20 | 刚接触 OpenClaw，基础交互 |
| Lv.2 | 🎮 初级玩家 | 21-40 | 开始使用自定义技能 |
| Lv.3 | 💻 中级开发者 | 41-60 | 多技能协作，自动化场景 |
| Lv.4 | 🚀 高级工程师 | 61-80 | 多 Agent，复杂工作流 |
| Lv.5 | 🦞 龙虾大师 | 81-100 | 全方位自动化，Proactive 模式 |

**评分维度**：

| 维度 | 权重 | 计算规则 |
|------|------|----------|
| 技能得分 | 50% | 技能数量 × 5 + 自定义技能 × 10 |
| 自动化得分 | 30% | 心跳开启 +20，定时任务数 × 5 |
| 集成得分 | 20% | 渠道数 × 10 |

#### 2.1.3 价值估算

```
价值 = 基础价值 + 技能价值 + 自动化价值
- 基础价值: 100元
- 技能价值: 技能数 × 200元
- 自动化价值: 心跳开启 × 500 + 定时任务 × 100
```

#### 2.1.4 成就系统

| 成就 | 条件 | 稀有度 |
|------|------|--------|
| 🏅 技能大师 | 自定义技能 ≥ 5 | 稀有 |
| 🛠️ 自定义大师 | 自定义技能 ≥ 7 | 史诗 |
| 🤖 自动化达人 | 心跳开启 | 普通 |
| 🌐 多渠道运营 | 渠道 ≥ 2 | 稀有 |
| ⚡ 超级用户 | 日均 Token ≥ 50000 | 传说 |

#### 2.1.5 趣味化表达

按等级生成不同风格的调侃文案，激发用户分享欲望：

- **Lv.1**: 幽默调侃，鼓励探索
- **Lv.2**: 积极引导，提示进阶
- **Lv.3**: 专业认可，展示实力
- **Lv.4**: 赞赏有加，展望大师
- **Lv.5**: 顶礼膜拜，传说级别

### 2.2 API 服务需求

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/` | GET | 仪表盘主数据 |
| `/api/stats` | GET | 统计数据 |
| `/api/skills` | GET | 技能列表 |
| `/api/sessions` | GET | 会话历史 |
| `/api/evaluation` | GET | 评估结果 |
| `/api/refresh` | POST | 刷新数据 |
| `/api/share-card` | POST | 生成分享卡片 |
| `/api/generate-image` | POST | 生成龙虾图片 |
| `/api/health` | GET | 健康检查 |

### 2.3 前端展示需求

- **主题系统**: 15 套风格迥异的主题
- **响应式布局**: 适配桌面和移动端
- **快捷键支持**: R/S/T/E/G/? 等快捷操作
- **数据可视化**: 技能分布饼图、能力雷达图
- **图片生成**: AI 生成专属龙虾形象
- **分享功能**: 生成分享卡片，展示龙虾等级

## 四、技术架构

### 4.1 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Python 3 + Flask |
| 数据库 | SQLite |
| 前端 | HTML5 + CSS3 + JavaScript |
| 图表 | Chart.js |
| 图片生成 | 阿里云万象 API |

```

├── docs/                    # 文档
│   ├── INTRO.md             # 项目介绍（本文档）
│   ├── REQUIREMENTS.md      # 需求文档
│   ├── SPEC.md              # 设计说明
│   ├── THEMES.md            # 主题设计规范
│   └── OPENCLAW_REFERENCE.md # OpenClaw 参考
├── lib/                     # 核心库
│   ├── constants.py         # 常量定义
│   ├── schemas.py           # 数据模型
│   ├── collector.py         # 数据采集
│   ├── evaluation.py        # 评估引擎
│   └── image_generator.py   # 图片生成
├── scripts/
│   ├── collector.py         # 命令行采集脚本

│   └── server.py            # Flask 服务
├── web/
```

## 五、快速开始

### 5.1 安装依赖

```bash
pip install -r requirements.txt
```

### 5.2 启动服务

```bash
python3 scripts/server.py --port 5002
```

### 5.3 访问地址

- 主页: http://localhost:5002
- API: http://localhost:5002/api/
