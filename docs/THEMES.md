# ClawValue 主题设计 SPEC

## 设计理念

ClawValue 的视觉设计目标是打造一个 **有趣、炫酷、具备爆款属性** 的龙虾值评估工具。

### 核心原则

1. **视觉冲击力** - 第一眼就要吸引眼球
2. **社交分享性** - 用户愿意截图分享
3. **个性化定制** - 多种风格满足不同用户
4. **动效沉浸感** - 流畅的动画增强体验

## 视觉元素规范

### 配色方案

```css
/* 深海龙虾 - 默认主题 */
--bg-primary: #0f0f23;      /* 主背景 */
--bg-secondary: #1a1a2e;    /* 次背景 */
--bg-card: #16213e;         /* 卡片背景 */
--text-primary: #e0e0e0;    /* 主文字 */
--text-secondary: #a0a0a0;  /* 次文字 */
--accent-primary: #ff6b6b;  /* 主强调色 */
--accent-secondary: #4facfe;/* 次强调色 */
--accent-highlight: #feca57;/* 高亮色 */
```

### 字体规范

- **标题**: Noto Serif SC, serif
- **正文**: -apple-system, sans-serif
- **数字**: SF Mono, monospace (等宽字体显示数值)

### 间距规范

- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px

### 圆角规范

- **sm**: 8px
- **md**: 12px
- **lg**: 16px
- **full**: 9999px

## 动效规范

### 进入动画

```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
```

### 卡片悬浮

```css
.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 20px 40px rgba(0, 0, 0, 0.3);
}
```

### 数字弹跳

```css
@keyframes numberPop {
  0% { transform: scale(1); }
  50% { transform: scale(1.2); }
  100% { transform: scale(1); }
}
```
