# ClawValue 主题系统 v3.0 - 扩展设计规格

> 创建时间: 2026-03-14
> 版本: v3.0
> 状态: 设计中

---

## 1. 设计目标

- 扩展到 **10 套风格迥异的主题**，覆盖更广泛的用户偏好
- 每套主题具有**独特的视觉语言**和**情感氛围**
- 主题切换**流畅自然**，有过渡动画
- 支持**主题预览**功能

---

## 2. 新增主题定义

### 主题 6: 🌙 暗夜森林 (Dark Forest)
- **风格**: 神秘自然风
- **关键词**: 月光、森林、神秘、宁静
- **配色**:
  - 背景: `#0d1b2a` → `#1b263b` 渐变
  - 主色: `#52b788` (森林绿)
  - 辅色: `#40916c` (松绿)
  - 强调: `#d4a373` (月光金)
  - 文字: `#e0e1dd` / `#778da9`
- **特点**: 
  - 星空粒子背景
  - 萤火虫浮动效果
  - 树叶边框装饰

### 主题 7: 🏖️ 夏日海滩 (Summer Beach)
- **风格**: 清爽度假风
- **关键词**: 阳光、沙滩、海洋、活力
- **配色**:
  - 背景: `#fefae0` → `#dda15e` 渐变
  - 主色: `#606c38` (棕榈绿)
  - 辅色: `#283618` (深绿)
  - 强调: `#bc6c25` (日落橙)
  - 文字: `#283618` / `#606c38`
- **特点**:
  - 波浪动画背景
  - 沙滩纹理
  - 阳光光晕效果

### 主题 8: 🍬 糖果乐园 (Candy Land)
- **风格**: 可爱甜美风
- **关键词**: 糖果、粉色、可爱、梦幻
- **配色**:
  - 背景: `#fff0f5` → `#ffb6c1` 渐变
  - 主色: `#ff69b4` (粉红)
  - 辅色: `#ff1493` (深粉)
  - 强调: `#87ceeb` (天蓝)
  - 文字: `#db7093` / `#c71585`
- **特点**:
  - 圆润边角 (32px+)
  - 柔和阴影
  - 糖果渐变按钮
  - 爱心浮动动画

### 主题 9: 🦇 暗黑哥特 (Dark Gothic)
- **风格**: 暗黑神秘风
- **关键词**: 哥特、暗黑、优雅、神秘
- **配色**:
  - 背景: `#0a0a0a` → `#1a0a0a` 渐变
  - 主色: `#8b0000` (血红)
  - 辅色: `#4a0a0a` (暗红)
  - 强调: `#c0c0c0` (银灰)
  - 文字: `#c0c0c0` / `#808080`
- **特点**:
  - 烛光闪烁效果
  - 蕾丝边框纹理
  - 滴血动画效果

### 主题 10: 💜 蒸汽波 (Vaporwave)
- **风格**: 复古未来风
- **关键词**: 80s、霓虹、故障、迷幻
- **配色**:
  - 背景: `#0d0221` → `#2d1b4e` 渐变
  - 主色: `#ff71ce` (粉紫)
  - 辅色: `#01cdfe` (青蓝)
  - 强调: `#05ffa1` (荧光绿)
  - 文字: `#ffffff` / `#b967ff`
- **特点**:
  - 网格背景
  - 故障艺术效果
  - 霓虹发光
  - 日落渐变
