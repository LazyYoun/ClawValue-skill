# 🎨 ClawValue 主题设计 SPEC v7.0

> 设计目标：打造具有爆款潜力的视觉体验，让用户第一眼就被吸引，愿意截图分享

## 📋 设计原则

### 核心原则
1. **视觉冲击力** - 第一眼就要惊艳
2. **社交分享性** - 截图即名片，自带话题
3. **讨论话题性** - 每套主题都有故事
4. **用户粘性** - 让用户想收集所有主题

- [x] 强烈的视觉对比
- [x] 独特的氛围感
- [x] 截图友好（数据清晰可见）
- [x] 社交货币属性（炫耀价值）
- [x] 话题讨论点

## 🌈 新增主题方案 (v7.0)

### 特色系 - 新增 5 套爆款主题

#### 1. 🌌 星河璀璨 (Galaxy Splendor)
**定位**: 宇宙探索 · 神秘浪漫
**目标用户**: 科技爱好者、浪漫主义者

**配色方案**:
```css
--bg-primary: #0a0a1a;        /* 深邃夜空 */
--bg-secondary: #1a1a3a;       /* 星空蓝 */
--bg-card: rgba(20, 20, 50, 0.85);
--accent-primary: #ff6b9d;     /* 粉红星云 */
--accent-secondary: #c084fc;   /* 紫色星尘 */
--accent-highlight: #22d3ee;   /* 青色星光 */
--text-primary: #f0f0ff;
--text-secondary: #a0a0c0;
```

**特效**:
- 星星闪烁动画 (动态生成)
- 流星划过效果
- 星云渐变背景
- 鼠标移动产生星尘轨迹

**话题点**: "你的龙虾值在宇宙中排名前 X%"

## 📝 技术实现要点

### CSS 变量扩展
```css
/* 新增特效变量 */
--effect-stars: true/false;
--effect-aurora: true/false;
--effect-rain: true/false;
--effect-snow: true/false;
--particle-color: #fff;
```

### JavaScript 粒子系统
```javascript
// 主题特效配置
const themeEffects = {
  'galaxy': { stars: true, shootingStars: true },
  'aurora': { aurora: true, snow: true },
  'neon-city': { rain: true, scanlines: true },
  'diamond': { sparkles: true, shine: true }
};
```

### 性能优化
- 移动端禁用复杂动画
- 使用 CSS will-change 优化
- requestAnimationFrame 控制粒子
