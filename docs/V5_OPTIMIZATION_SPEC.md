# ClawValue v5.0 视觉优化 SPEC

## 目标
让页面从"功能完备"进化到"惊艳吸睛"，具备爆款传播属性。

## 设计原则
1. **第一眼惊艳** - 3秒内抓住用户眼球
2. **差异化主题** - 每套主题有独特的视觉语言
3. **微交互** - 细节处处有惊喜
4. **社交传播** - 分享即炫耀

## 优化模块

### 1. 英雄区域重设计 (Hero Section)
- 动态粒子背景（龙虾主题）
- 渐变流动标题
- 实时数据动态展示
- CTA 按钮发光脉动

### 2. 卡片系统升级
- 毛玻璃效果增强
- 悬浮时 3D 倾斜效果
- 数据加载时的骨架屏动画
- 卡片呼吸动画

### 3. 主题差异化
每套主题增加独特的：
- 背景动效（气泡/粒子/光线）
- 转场动画
- 按钮风格
- 字体搭配

### 4. 互动增强
- 鼠标跟随光晕
- 数字滚动动画
- 图表入场动画
- 成就解锁特效

### 5. 社交属性
- 分享卡片动态生成
- 排行榜功能
- 勋章墙展示

## 技术实现
- CSS 变量扩展
- Keyframes 动画库
- Intersection Observer
- CSS Grid 高级布局
